# [qier222.com](https://qier222.com)

My personal website.

### Run

```bash
git clone https://github.com/qier222/qier222.com.git
cd qier222.com
npm run install
cp .env.example .env # then edit .env
npm run dev
```

### Built with

- [Next.js](https://nextjs.org)
- [Vercel](https://vercel.com)
- [Supabase](https://supabase.io)
